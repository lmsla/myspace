# Elasticsearch啟動相關指令



| 功能        | 指令                              | 
| :--------: | --------------------------------- | 
| 啟動            | $ systemctl start elasticsearch   |   
| 開機自動啟動 | $ systemctl enable elasticsearch |    
| 停止       | $ systemctl stop elasticsearch    | 
| 狀態查詢     | $ systemctl status elasticsearch  | 
| 重新啟動     | $ systemctl restart elasticsearch  |
